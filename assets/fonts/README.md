# Fuentes Tipográficas

## Bubbleboddy Neue

Fuente principal para títulos y headers.

**Archivos incluidos:**
- `BubbleboddyNeue-Regular.ttf` - Peso regular (400)
- `BubbleboddyNeue-Bold.ttf` - Peso bold (700)

**Uso:** Títulos, nombres, headers principales

## Garet

Fuente para cuerpo de texto y acentos.

**Archivos incluidos:**
- `Garet-Book.ttf` - Peso book (400)

**Uso:** Cuerpo de texto, acentos, etiquetas, labels

## Configuración

Ambas fuentes se cargan localmente desde `src/index.css` usando `@font-face`.

No se requieren fuentes externas ni Google Fonts.
